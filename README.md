# AgenticVidura
An agentic architecture for science gateways that uses reflection loops, tool calling, and iterative reasoning to support reliable, traceable knowledge discovery.

Vidura 2.0 is an agentic system designed for reflection-driven knowledge discovery in science gateways. It combines planning, tool use, and iterative refinement to produce traceable outputs, reduce hallucinations, and support reproducible workflows across scientific tasks.
